# Dokumente-Modul — UI-Plan

> Stand: 2026-04-03 · Session: eeee-dokumente

## Sidebar

```
📄 Dokumente          ← Neuer Hauptpunkt
   ├── Eingang        ← Neue/unbearbeitete Dokumente
   ├── Zugeordnet     ← Einem Vorgang zugeordnet
   ├── Studio         ← Erstellen/Bearbeiten
   ├── Vorlagen       ← Vorlagen-Verwaltung
   └── Archiv         ← Abgeschlossene Dokumente
```

Position: Nach Kommunikation, vor Organisation.

## Dokumenten-Liste (Eingang/Zugeordnet/Archiv)

3-Spalten wie Postfach:
- Links: Filter/Navigation (Quelle, Typ, Vorgang, Status)
- Mitte: Dokumenten-Liste (Icon, Name, Datum, Quelle, Status, Vorgangsbezug)
- Rechts: Preview + Metadaten + Aktionen

## Dokument-Studio UI

3-Spalten:
- Links: Vorlagen-Browser, Letzte Entwürfe, Dokumentquellen
- Mitte: Quill-Editor / PDF-Preview / DOCX-Preview
- Rechts: Variablen-Panel, Vorgang-Bezug, Layout-Optionen, Export-Buttons

## Einstellungen-Sektion

Unter "Dokumente" in Einstellungen:
- Überwachter Ordner (Pfad, Unterordner, Intervall)
- Dateitypen (Checkboxen)
- OCR (Aktiv/Sprache)
- Externe Ablage (Pfad, Ordnerstruktur)
- Dublettenerkennung (An/Aus)
- Kira-Regeln (darf erstellen, darf zuordnen)
- Modul-Status (Feature-Flag)
